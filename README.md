# Sales Data Analysis using Python and Pandas

Project Overview
This project demonstrates basic data analysis using Python and the Pandas library.  
The goal is to analyze a simple sales dataset, clean missing values, calculate key metrics, and generate meaningful insights.

This project is designed for beginners to understand how real-world data is handled in Python.

---

Objectives
- Load and analyze CSV data using Pandas
- Explore dataset structure and data types
- Handle missing values
- Calculate sales metrics
- Generate a simple analysis report

---

Dataset Information
**File:** `sales_data.csv`

**Columns:**
- Order_ID
- Product
- Quantity
- Price
- Total_Sales

**Rows:** 10  
The dataset contains missing values in Quantity and Total_Sales columns, which are handled during data cleaning.

---

Technologies Used
- Python 3
- Pandas library
- CSV file format
- Git & GitHub
