# Sales Data Analysis Report

## Project Overview
This project analyzes a simple sales dataset using Python and Pandas to understand revenue, product performance, and sales trends.

## Objectives
- Load and clean sales data
- Handle missing values
- Calculate key metrics
- Generate insights

## Dataset Details
- Rows: 10
- Columns: 5
- Columns: Order_ID, Product, Quantity, Price, Total_Sales

## Data Cleaning
- Missing quantities filled with 0
- Total_Sales calculated where missing
- Duplicate rows removed

## Analysis Results
- Total Revenue: ₹510,000
- Best Selling Product: Laptop
- Highest Sale: ₹100,000
- Lowest Sale: ₹0
- Average Sale: ₹51,000

## Conclusion
The Laptop is the top revenue-generating product. Data cleaning was essential to ensure accurate results.

## Tools Used
- Python
- Pandas
